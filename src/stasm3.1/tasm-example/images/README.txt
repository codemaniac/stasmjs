README for tasm-example/images
------------------------------

The faces in this directory are from the BioID data set
www.bioid.com/downloads/facedb/index.php

Reference:
O. Jesorsky, K. Kirchberg, R. Frischholz.
Robust Face Detection Using the Hausdorff Distance
In J. Bigun and F. Smeraldi, editors, Audio and Video based Person
Authentication - AVBPA 2001, pages 90-95. Springer, 2001.
